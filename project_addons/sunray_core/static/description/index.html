Sunray Core - WebAuthn Authentication System

Secure, passwordless authentication for Cloudflare Workers

Features

  • WebAuthn/Passkey authentication
  • JWT-based session management
  • Multi-host support
  • CIDR whitelist bypass
  • Public URL patterns
  • Webhook token management
  • Comprehensive audit logging
  • Setup token generation with QR codes

Architecture

Sunray consists of two main components:

  • Admin Server (this module): Odoo 18 addon for configuration and management
  • Cloudflare Worker: Edge authentication and request handling

Security Model

  • Default locked: All resources protected by default
  • Whitelist exceptions for trusted IPs and public URLs
  • WebAuthn/Passkeys for passwordless authentication
  • JWT signatures with HMAC-SHA256
  • Device fingerprinting for session binding
  • CSRF protection with double-submit cookies
